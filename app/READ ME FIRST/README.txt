Patient Management System (React Native / Expo)

A mobile app that allows patients to create cases and doctors to view, manage, and provide consultations. 
built using React Native with Expo, featuring local storage using AsyncStorage and a modular screen/component structure.

THIS APPLICATIONS PROVIDES THE FOLLOWING FEATURES:
- Patient registration & case creation
- Nurse case management
- Local storage for patient data
- Simple navigation between screens
- Minimal UI, easy to customize
